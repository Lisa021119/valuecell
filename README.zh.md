# InsightDesk（ENMGT 5400 课程演示）

**InsightDesk** 为本课程使用的本地投研与多智能体演示应用，默认在本地运行，不依赖外部商业产品地址（除非自行配置环境变量）。

## 运行

```bash
bash start.sh
```

- 浏览器打开：[http://localhost:1420](http://localhost:1420)  
- 后端 API：[http://localhost:8000](http://localhost:8000)  

配置 API Key 等请见 [docs/CONFIGURATION_GUIDE.md](docs/CONFIGURATION_GUIDE.md)。
