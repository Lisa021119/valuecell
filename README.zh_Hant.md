# InsightDesk（ENMGT 5400 課程演示）

**InsightDesk** 為本課程使用之本地投研與多智慧體展示應用，預設在本機執行，不連線至外部商業產品（除非自行設定環境變數）。

## 執行

```bash
bash start.sh
```

- 瀏覽器開啟：[http://localhost:1420](http://localhost:1420)  
- 後端 API：[http://localhost:8000](http://localhost:8000)  

金鑰與資料源設定見 [docs/CONFIGURATION_GUIDE.md](docs/CONFIGURATION_GUIDE.md)。
